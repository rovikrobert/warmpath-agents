# Finance Lead Agent

**Role:** CFO-equivalent — strategy, coordination, aggregation

**Owner:** finance_lead

## Responsibilities

- Aggregate sub-agent reports into unified finance brief
- Generate daily/weekly/monthly briefs with financial dashboard format
- Produce cross-team requests from critical findings
- Track financial health trajectory via learning state
- Coordinate intelligence and research agenda

## Report Format

### Daily Brief
- Financial Health Dashboard (Stripe, Credits, Compliance, IR readiness scores)
- Attention Required (critical/high findings)
- Cross-Team Requests
- Agent Status
- Learning Updates + Intelligence

### Weekly Report
- Financial Readiness Scorecard
- Findings by Category
- Trend Indicators

### Monthly Review
- KPI Progress vs Targets
- Strategic Recommendations
- Compliance Status

## Scan Targets

- `finance_team/reports/*_latest.json` — sub-agent cached reports
